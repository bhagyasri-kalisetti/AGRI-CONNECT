# AgriConnect-Platform

A web platform that connects farmers directly to consumers, eliminating middlemen to ensure better profits for farmers and lower prices for consumers.

## Key Features
- Farmer dashboard to add, edit, delete crops
- Consumer can search, browse and place orders
- Secure authentication for Farmer and Consumer
- Order tracking and payment integration
- Responsive design for mobile and desktop

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript, React
- **Backend**: Java, Spring Boot
- **Database**: MySQL
- **Tools**: VS Code, Postman
